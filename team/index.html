The people behind the software

Team

Small team, deep specs, real servers. New contributors are welcome.

NK

Nicco Kunzmann

@niccokunzmann

Nicco holds this together. He maintains more packages than anyone else, reviews code across the whole ecosystem, brought most of this team together, and is the primary contact for NLnet. The coordinated effort you see here is largely his doing.

icalendar recurring-ical-events open-web-calendar ical2jcal ics-query x-wr-timezone
TB

Tobias Brox

@tobixen

The original. Tobias started building CalDAV tooling in 2013 because he needed it, not because it was a project. That work became calendar-cli, then python-caldav, then plann. Over a decade later he still maintains all three and knows where every server quirk is buried.

python-caldav plann calendar-cli
SB

Sashank Bhamidi

@SashankBhamidi LinkedIn

Sashank is the Founder of BizCred and does open source on the side. He built JMAP calendar support into python-caldav from scratch, and maintains icalendar and icalendar-anonymizer.

icalendar icalendar-anonymizer
AH

Abe Hanoka

@abe-101 LinkedIn

Abe runs MergeCal.org and MyHebrewDates.com on these packages, which means he hits the bugs that only show up at scale. He maintains python-mergecal and django-icalendar, and is also a maintainer of dj-stripe.

python-mergecal django-icalendar
SP

Steve Piercy

@stevepiercy

Steve leads docs for this project and for Plone, is bringing iCalendar support to Plone's event content types, and maintains sphinxcontrib-httpdomain, used across the Python documentation community for REST API docs.

Documentation sphinxcontrib-httpdomain
