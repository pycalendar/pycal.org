The Python Web Calendaring Ecosystem

Calendaring is harder than it looks. We do it properly.

Browse software About the project

RFC 5545, CalDAV, JMAP: each one has quirks, and real servers pile on more. Most libraries quietly get recurrences wrong or skip the hard parts entirely. We put an effort into extending compatibity.

Small team, real servers, no company behind it. Funded by NLnet NGI Zero. Parts of this software run in millions of installations.

Key software

icalendar Stable

RFC 5545 parser and generator for .ics files. The foundation every other package in this ecosystem builds on.
python-caldav Stable

The only actively maintained Python CalDAV client. Sync and async, scheduling, free/busy, and JMAP. Used directly by HomeAssistant.
recurring-ical-events Stable

Expands recurring events, TODOs, journals, and alarms for a given date range. Used by Nextcloud, Thunderbird, and Android.
open-web-calendar Stable

Turns iCalendar feeds into an embeddable web calendar. No account, no database, no tracking. Self-hostable.
icalendar-anonymizer Stable

Strips personal data from .ics files while preserving all technical properties. Library, CLI, HTTP API, and web UI.
plann Stable

CLI and TUI for managing calendars and tasks directly on a CalDAV server. No local sync required.
