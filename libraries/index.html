Python Calendaring Ecosystem

Libraries

All packages maintained by this team. A library can appear in more than one category. Badges show stability status.

Utility libraries

icalendar Stable

Reads and writes .ics files. The base every other library builds on.
recurring-ical-events Stable

Give it a date range, get back every event in it. Handles recurring events, exceptions, and overrides correctly.
icalendar-anonymizer Stable

Scrubs personal data out of .ics files without breaking the structure. Available as a Python library, CLI, HTTP API, or web UI.
icalendar-searcher Stable

Filter, sort, and expand calendar components on the client side. Useful when your server's query support doesn't cover what you need.
python-mergecal Stable

Merges multiple iCalendar feeds into one. Running in production at MergeCal.org and MyHebrewDates.com.

Client/server communication

python-caldav Stable

The only actively maintained Python CalDAV client. Sync and async, scheduling, free/busy, JMAP. Tested against Nextcloud, Baikal, Cyrus, SOGo, Fastmail, Google Calendar, and others.
calendaring-jmap Not implemented

The future standalone Python JMAP calendar library. The implementation currently lives in python-caldav and will be extracted here.
calendaring-client Not implemented

One API for CalDAV, JMAP, and raw .ics files. Point it at a URL and it figures out what it's talking to.

open-web-calendar Stable

Turns iCalendar feeds into a calendar you can view in any browser. No account, no database, no tracking. Configure with URL parameters and embed anywhere.

Applications

plann Stable

CLI and TUI for managing calendars and tasks. Talks directly to your CalDAV server, no local sync. Successor to calendar-cli.
open-web-calendar Stable

Turns iCalendar feeds into a calendar you can view in any browser. No account, no database, no tracking. Configure with URL parameters and embed anywhere.
calendar-cli Sunset

The original CalDAV CLI. Still maintained for backward compatibility. New users should use plann instead.

Integrations

django-icalendar Proof of concept

Django models for storing and exporting RFC 5545 iCalendar data. Brings CalendarModel, EventModel, forms, and admin integration to Django projects.
