Python Web Calendaring Ecosystem

Software

Everything we maintain. Badges show stability status.

Core

icalendar Stable

RFC 5545 parser and generator for .ics files. The foundation everything else in this ecosystem builds on.
python-caldav Stable

The only actively maintained Python CalDAV client. Sync and async, scheduling, free/busy, JMAP. Tested against a dozen real servers. Used by HomeAssistant.
recurring-ical-events Stable

Give it a calendar and a date range, get back every occurrence — exceptions and overrides included. Used by Nextcloud, Thunderbird, and Android.

Utilities

icalendar-anonymizer Stable

Strip names, emails, and locations from .ics files while keeping all timing and recurrence data intact. Library, CLI, HTTP API, web UI, and Docker image.
icalendar-searcher Stable

Client-side search and filtering for iCalendar data — for when the server doesn't support the query you need.
python-mergecal Stable

Merge multiple iCalendar feeds into one. Handles Google Calendar's timezone quirks, deduplicates events. Running in production at MergeCal.org and MyHebrewDates.com.
x-wr-timezone Stable

Fixes the non-standard X-WR-TIMEZONE property that Google Calendar exports, so .ics files work correctly in strict parsers.
ics-query Stable

Query .ics files by time range and component type from the command line. Outputs valid .ics. Available via Homebrew, Windows .exe, or pip.
ical2jcal Stable

Converts between iCalendar (RFC 5545, .ics) and jCal (RFC 7265, JSON) formats. Available as a Python library and CLI.

Client/server communication

calendaring-jmap Planned

JMAP calendar support as a standalone package — scheduling, free/busy, attachments, push notifications. Currently part of python-caldav, being extracted here.
calendaring-client Planned

One API for CalDAV, JMAP, and plain .ics feeds. Point it at a URL — it figures out the rest.

Applications

open-web-calendar Stable

Embed any iCalendar feed as a web calendar. No account, no database, no tracking. Self-hostable.
plann Stable

CLI and TUI for calendars and tasks, live against a CalDAV server. No local sync, no local storage. Successor to calendar-cli.
calendar-cli Sunset

The original CalDAV CLI. Still works, still maintained. New users should start with plann.

Tooling

caldav-server-tester Stable

Point it at a CalDAV server and get a full report of what it actually supports — not just what the docs claim.
