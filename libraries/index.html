Python Calendaring Ecosystem

Libraries

All packages maintained by this team. A library can appear in more than one category. Badges show stability status.

Core libraries

icalendar Stable

RFC 5545 compatible parser and generator for iCalendar (.ics) files. The foundation everything else in this ecosystem builds on. Supports zoneinfo, dateutil, and pytz timezones. Used by python-caldav, recurring-ical-events, open-web-calendar, and many other projects.
python-caldav Stable

The only actively maintained Python CalDAV client library. Supports both sync and async usage, scheduling (RFC 6638), free/busy queries, managed attachments, and JMAP calendar access. Tested against Nextcloud, Baikal, Cyrus IMAP, SOGo, Fastmail, Google Calendar, Stalwart, and others. Used directly by HomeAssistant.
recurring-ical-events Stable

Calculates recurrence times for events, TODOs, journal entries, and alarms from iCalendar data. Give it a calendar and a date range, get back every occurrence — including exceptions and overrides. Used by Nextcloud, Thunderbird, Android, and Eventyay.

Utility libraries

icalendar-anonymizer Stable

Strips personal data from .ics files using hash-based anonymization — names, emails, locations, and descriptions are replaced while date/time, recurrence patterns, and timezone data are preserved intact. Available as a Python library, CLI, HTTP API, web UI, and Docker image.
icalendar-searcher Stable

Client-side search, filtering, and sorting of iCalendar components. Handles property-based filtering, date range queries, component type selection, category matching, and locale-aware collation. Fills in when servers lack support for specific queries.
python-mergecal Stable

Merges multiple iCalendar feeds and .ics files into a single calendar. Handles Google Calendar's non-standard X-WR-TIMEZONE property, deduplicates events, and tracks additions and deletions across source feeds. Available as both a Python library and CLI. Running in production at MergeCal.org and MyHebrewDates.com.
x-wr-timezone Stable

Fixes calendars exported by Google Calendar and similar tools that use the non-standard X-WR-TIMEZONE property. Converts them to RFC 5545-compliant format so they display correctly in strict parsers. Available as a Python library and CLI; supports piping with wget or curl.

ics-query Stable

Query and filter RFC 5545 iCalendar files by time range, component type (VEVENT, VTODO, VJOURNAL, VALARM), and timezone. Outputs valid .ics files. Available as a Python library, CLI, Windows .exe, and via Homebrew on macOS. Supports piping and concatenating multiple calendar files.
ical2jcal Stable

Converts between iCalendar (RFC 5545, .ics) and jCalendar (RFC 7265, JSON) formats. Available as a Python library and CLI.

Client/server communication

calendaring-jmap Planned

A standalone Python JMAP calendar library. JMAP calendar support — scheduling, sharing, free/busy, attachments, contacts integration, and push notifications — currently lives in python-caldav and will be published here as an independently installable package.
calendaring-client Planned

A unified Python API for CalDAV, JMAP, and raw iCalendar feeds. Point it at a URL and it detects what it's talking to — no need to know whether the backend is CalDAV, JMAP, or a plain .ics file. First-class task support alongside events.

Applications

open-web-calendar Stable

Turns iCalendar feeds into a customizable calendar you can embed in any website. No account, no database, no tracking. Supports multiple sources, per-calendar styling, day/week/month/agenda views, multi-language, and timezone selection. Self-hostable via Docker, Python package, or direct deployment.
plann Stable

CLI and TUI for managing calendars and tasks directly against a CalDAV server — no local sync, no local storage. Add events, browse agendas, manage todos, postpone tasks, operate across multiple calendars simultaneously. Supports JSON/YAML config, custom output templates, and explicit timezone handling. Successor to calendar-cli.
calendar-cli Sunset

The original CalDAV CLI, maintained for backward compatibility. Mature and production-ready. New users should start with plann instead.

Tooling

caldav-server-tester Stable

Probes a CalDAV server and reports which features and RFC requirements it supports. Checks calendar creation, event/task/journal persistence, time-range search, text search, synchronization reports (RFC 6578), free/busy queries, principal discovery (RFC 5397), UID handling, and timezone support. Outputs JSON, YAML, or compatibility hints.

Integrations

django-icalendar Proof of concept

Django models for storing and exporting RFC 5545 iCalendar data. Provides CalendarModel and EventModel with dual date/datetime field patterns, recurrence rule storage, organizer and attendee support, and Django admin integration.
calendaring-client Planned

A single consistent API for applications needing to talk to CalDAV servers, JMAP servers, or read plain .ics files — without knowing which backend is behind a URL.
